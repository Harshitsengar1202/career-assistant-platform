# AI-Powered Autonomous Job Application & Career Assistant Platform

This package contains a complete starter delivery set derived from the SRS:

- `Enterprise_SRS_v2.1.docx` and `.pdf`: polished enterprise requirements document.
- `architecture.md`: system, agent, data, and deployment diagrams in Mermaid.
- `user_stories.md`: epics, user stories, and acceptance criteria.
- `api/openapi.yaml`: REST API contract starter.
- `database/schema.sql`: PostgreSQL schema baseline.
- `project_plan.xlsx`: MVP roadmap, backlog, risks, and milestone summary.
- `starter-app/`: runnable frontend prototype plus FastAPI backend scaffold.

The starter app is intentionally lightweight: the frontend can be opened directly in a browser, and the backend is a clean FastAPI skeleton ready for implementation.
